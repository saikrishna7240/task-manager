Task Manager Requirements
1. Project Objective

Build a secure task management web application where users can create, organize, update, and track their daily tasks.

2. Target Users
Students
Employees
Freelancers
Anyone who wants to manage tasks
3. User Features
Authentication
Register
Login
Logout
Change password
Update profile
Task Management
Create a task
View all tasks
View task details
Edit a task
Delete a task
Mark a task as completed or pending
Task Details

Each task contains:

Title
Description
Category
Priority
Due date
Status
Created date
Updated date
Search & Filters
Search by title
Filter by category
Filter by priority
Filter by status
Sort by due date
Sort by newest/oldest
Dashboard

Show:

Total tasks
Completed tasks
Pending tasks
Overdue tasks
Completion percentage
4. Non-Functional Requirements
Responsive on desktop and mobile
Fast loading
Secure password storage
JWT-based authentication
Input validation
Good error messages
5. Technology Stack
Frontend: React
Backend: Node.js + Express
Database: SQL
Authentication: JWT
Password hashing: bcrypt
6. Project Scope

Included

User authentication
Task CRUD
Dashboard
Search and filters
Responsive design

Not included (Version 1)

Email notifications
Team collaboration
Calendar sync
Push notifications
7. Success Criteria

A user should be able to:

Create an account
Log in securely
Manage tasks
Search and filter tasks
View task statistics
Use the application smoothly on desktop and mobile
